# Web Research Skill

Use this skill to gather information from external web sources.

## When to Use

- Researching external documentation
- Gathering information from URLs
- Scraping web content for analysis
- Learning about external libraries or frameworks
- Validating information from live sources

## Guidelines

1. **Respect rate limits** - Avoid excessive requests to the same domain
2. **Validate URLs** - Ensure URLs are properly formatted before scraping
3. **Handle errors gracefully** - Some URLs may be inaccessible or blocked
4. **Use for external resources** - Leverage web research for information outside the project
5. **Combine with file management** - Use file tools to save research findings to project files

## Available Tools

- **scrapeUrls** - Scrape content from one or more URLs
